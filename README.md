# dr-jam




Dr. JAM prompt:

You will build a static webpage for my grandfather and his music. His name was John Andrew Moreland and his nickname was "Dr. JAM". The website should be easy to navigate and simple. The style should be clean but not super modern. This is a very family oriented type of website so the aesthetic should look very friendly. It should look good and work well on mobile as well as desktop.

For colors I'm thinking softer tones. Nothing extermely high contrast. It needs to be friendly, inviting, and family oriented.
For vibe I'm thinking "old photo". I'm picturing old photo texture with that kind of grainy-ness.


I would like the website tabs to be tabs and not just stacked vertically, so you cannot get to the different tabs without clicking on them. The top of the page should have a banner type thing which can have pictures in it. In the upper left it should say "Dr. JAM". The 3 different tabs should be cetrally located under the banner. Overally it will end up looking a bit like a wedding website in terms of the format.

You will create the following 3 tabs in the website :
- Bio tab with some pictures 
- tab for recorded CD's (album player style with WAV files)
- tab for video's or video links to performances of his arrangements

Bia Tab details:
For the BIO tab I will be providing a bio with pictures to go with it. For now just make up some text. At the top it should have "Dr. JAM" in big letters and then below it in smaller letters "John Andrew Moreland". It should also have his birth and death years which are 1926-2019.

Studio Recordings Tab details:
For the recordings tab there should be a big section type of thing for each album. All the tracks for the album should be listed and each one should be playable. If you play a track it should automatically go to the next track when it's finished playing. If the last track of the album finished playing though it should stop. We don't need shuffle buttons or anything like that. Let's keep it simple and do a play button for each track and a play all button for the whole album. Obviously we will need play and pause buttons. The albums are in the music folder in resources.

vidoes tab details:
We don't have any yet but this will be where you could find video recordings of his band or bands playing his arrangemnets. 


Other details:
- the browser tab should read "Dr. JAM" and it should have some little music note icons.
